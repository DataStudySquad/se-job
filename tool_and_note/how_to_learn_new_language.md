# 如果開始學一種新的程式語言

## 1. 環境建置
* 前端裝好瀏覽器
* 後端建議弄個Linux環境
* 裝個vscode
* 有Git, Github佳

## 2. 選教材
* 找官方網站的Get started內容
* 把w3shool的線上練習做一次
* 找一個`程式語言 by example`的教學練習

## 3. 做專案練習
>> 可用CLI代入輸入輸出值
* 做一個兩數相加的功能，用CLI代入輸入輸出值
* 用迴圈從1印到5
* 開根號練習
* 函式的練習 定義參數 函示 回傳值
* 練習陣列
* 做一個todo list  api server
