## 英文名詞 columns 和 rows 方向的記憶方法

* rows 就是水平的方向欄位(horizontal)
    * horizontal + rows -> HR (人資)
* columns 就是水平的方向欄位(vertical)
    * columns + vertical -> CV (履歷)
